Список команд

  • git init - створює новий лок. проект
  • git clone ... - скачує репозиторій з github
  • git status - перевіряє статус зміни створення комінітів
  • git branch- відображає інформацію про гілки в репозиторії
  • git push - вносить зміни у віддалений репозиторій
  • git pull - оновлює репозиторій локальний з віддаленого
